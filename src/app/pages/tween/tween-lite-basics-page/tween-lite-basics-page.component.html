Tween Basics

let tween = TweenLite.[method](target(s), duration, properties);

Method

Defines how the animation starts. Choices are .from, .to, and .fromTo

Target(s)

  • String will default to getElementById if string starts with '#', otherwise will use querySelectorAll
  • If you have jQuery installed as well, will use that as the query seelctor
  • Also accepts JavaScript DOM elements

Duration

How long will the animation last? Usually in seconds.

Properties

Can either be the properties to be animated, a number of other named options, or callbacks.

Notable Options include:
  • delay: Add a delay to when the animation starts
  • ease: The easing method to use
  • paused: Boolean if the animation should start on creation
Callbacks
  • onComplete
  • onStart
  • onUpdate
