GSAP (GreenSock Animation Platform)

Why GSAP?

Plenty of alternatives: CSS, jQuery, Velocity.js...

  • Longevity: started as a flash library
  • Documentation
  • Actively Maintained
  • Fast
  • No Dependancies
  • Fairly lightweight at 114kb for everything

What is GSAP?

Animation Platform: allows you to animate pretty much anything. Allows you to control several animations at the same time, and has a number of handy extensions.

The main components are Tween and Timeline. Lite and Max vareties of each.
