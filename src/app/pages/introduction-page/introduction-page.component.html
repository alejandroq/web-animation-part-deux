Why Web Animation?

Animation Orients the User and Gives Context

Creates spatial relationships between elements on a page and reduces cognitive load.

Animation Directs Focus and Draws Attention

Animated items draw focus and empasis importance and can direct eye flow.

Animation Shows Cause and Effect

Affordances suggest functionality, preview actions, and help correct mistakes.

Animation Provides Feedback

Errors can stand out more, as can success. Progress can be demonstrated as well.

Animation to Demonstrate

You can show how something works with immediate understanding.
